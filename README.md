
# CodeRefine – Generative AI Powered Code Review & Optimization Engine

CodeRefine is a simple AI-powered platform that analyzes source code and provides:
- Bug detection
- Performance suggestions
- Security warnings
- Optimized code suggestions

## Features
- Automated code review
- Performance optimization suggestions
- Basic security checks
- Structured AI analysis report

## Installation

git clone https://github.com/yourusername/coderefine-ai-code-review
cd coderefine-ai-code-review
pip install -r requirements.txt

## Run Backend

cd backend
python app.py

Then open frontend/index.html in your browser.
